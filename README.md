# Kitap Takip Uygulaması

Bu proje, kitaplarınızı takip etmek ve okuma hedeflerinizi düzenlemek için bir uygulamadır.

## Özellikler
- Okunan kitapları listeleme
- Okuma hedefi belirleme
- Kitaplara not ekleme ve düzenleme
